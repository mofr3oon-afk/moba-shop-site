# MOBA SHOP Upgrade V3

النسخة دي فيها:

- بيانات الدفع تظهر بعد اختيار طريقة الدفع فقط
- زر نسخ لكل رقم واسم ويوزر
- زر فتح لينك InstaPay
- رقم طلب داخلي للادمن فقط: MOBA 1001 ويتجدد يوميا
- العميل يتابع الطلب برقم الموبايل فقط
- اوامر ادمن: /today /week /month /customer /orders /pending /delivered
- زر سجل العميل تحت كل اوردر
- منع تكرار الطلب المفتوح لنفس الرقم ونفس PUBG ID
- Header عام للجيمرز مش محصور في ببجي
- كروت ثقة وخطوات الطلب
- كارت نجاح بعد الطلب
- ازرار مشكلة جاهزة: سكرين غير واضح / ID غلط

## تحديث V4
- إضافة خانة اسم حساب PUBG مع كل ID.
- حذف خانة الاسم الاختياري من الدفع.
- السماح بتكرار نفس PUBG ID في السلة أو الطلبات، لأن العميل ممكن يشحن كذا باقة لنفس الحساب.
- تحسين عرض الموبايل: كروت المنتجات جنب بعض + أزرار انتقال سريعة تحت الهيدر للسلة وحالة الطلب والدعم.

## تحديث V5
- تأكيد التحويل: العميل يختار هل التحويل من نفس رقم المتابعة أم من رقم / محل آخر.
- لو التحويل من رقم أو محل آخر، آخر 3 أرقام من رقم التحويل إلزامية ولا يتم الطلب بدونها.
- أزرار الادمن بها: سكرين غير واضح / ID غلط / رقم غلط.
- حالة الطلب تظهر للعميل زر تعديل مرة واحدة فقط حسب نوع المشكلة.
- بعد استخدام فرصة التعديل، لو المشكلة لم تُحل يمكن رفض الطلب ويظهر للعميل زر دعم تليجرام.
- أوامر وتقارير الادمن تعمل من /admin أو /reports مع أزرار: اليوم / الأسبوع / الشهر / الطلبات المفتوحة / تم الشحن اليوم.


## تحديث V6
- إضافة عداد كمية في السلة للمنتجات العادية مثل الشدات.
- عند زيادة الكمية يظهر إجمالي الشدات لهذا ID.
- الازدهار والكريستالة والبرايم تظل كمية 1 فقط لتجنب مشاكل القيود داخل اللعبة.
- تحسين قوي للهيدر على الموبايل وتصغيره وإخفاء الكروت غير المهمة.
- إضافة زر عائم صغير لمتابعة حالة الطلب.
- تفعيل أزرار الاختصار فوق للهيدر: المنتجات، السلة، حالة الطلب، الدعم.
